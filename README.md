#  My Shopping Store (JS E-Commerce)

A modern, lightweight Shopping Cart application built with Pure JavaScript (Vanilla JS), HTML5, and CSS3. This project demonstrates DOM manipulation, data persistence using LocalStorage, and real-time UI updates.

##  Key Features

- *Product Management:* Add new products with a name, price, and image URL.
- *Persistent Storage:* Uses LocalStorage to save your cart data so it remains available even after a page refresh.
- *Dynamic Search:* Real-time search functionality to filter products instantly as you type.
- *Cart Controls:* - Increase/Decrease quantity (+ / -).
  - Remove specific items from the cart.
  - Automatic subtotal and grand total calculations.
- *Modern Responsive UI:* A clean, grid-based layout that adapts to different screen sizes.

## 🛠️ Tech Stack

- *HTML5:* Semantic structure.
- *CSS3:* Custom styling, Flexbox/Grid layouts, and interactive hover effects.
- *JavaScript (ES6):* Logic handling, array methods (filter, map, splice), and Event Listeners.

##  Project Files

- index.html: The main entry point and structure.
- style.css: All the professional styling and layout rules.
- script.js: The engine of the app (Handling data and UI updates).

##  How to Run the Project

1. *Download/Clone* the repository to your local machine.
2. Open the index.html file in any modern web browser (Chrome, Firefox, Edge).
3. Start adding items! Enter a product name, price, and an image URL, then click *"Add Data"*.
4. Use the *Search Bar* to find specific items in your list instantly.

##  Technical Highlights

- *datacollection()*: The core function that dynamically renders product cards based on the current state or filtered results.
- *calculateTotal()*: Iterates through the cart array to provide an accurate real-time bill.
- *Logic Validation*: Prevents negative quantities and handles "Empty Cart" states gracefully.
